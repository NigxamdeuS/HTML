HTMLタグ一覧と概要

HTMLで使用される主なタグをカテゴリ別に整理した一覧です。基本的なページ構造を作るためのタグから、テキスト装飾、リスト、リンク、メディア埋め込み、フォーム作成、テーブル作成など、多岐にわたる要素を網羅しています。それぞれのタグの役割と用途が簡潔にまとめられており、初心者から中級者まで幅広いレベルの開発者に役立つ資料です。

基本タグ

!DOCTYPE html ドキュメントタイプを宣言。
html HTMLドキュメント全体を囲むルート要素。
head メタデータを定義（タイトルやCSS、スクリプトのリンクなど）。
title ページタイトルを設定。
meta メタ情報を指定（文字セット、ビューポートなど）。
link 外部リソース（CSSなど）をリンク。
style ページ内にCSSを記述。
script ページ内にJavaScriptを記述または外部スクリプトをリンク。

ページ構造

body ページの主要な内容を囲む。
header ページやセクションのヘッダー。
footer ページやセクションのフッター。
nav ナビゲーションリンクを囲む。
main 主要なコンテンツ領域。
section セクションを定義。
article 独立した記事やコンテンツ。
aside 補足的な情報。
div ブロックレベルの汎用コンテナー。

テキスト関連

h1～h6 見出しレベルを指定。
p 段落を定義。
span インライン要素の汎用コンテナー。
br 改行。
hr 水平線（区切り線）。
strong 強調（太字）。
em 強調（斜体）。
b 太字（強調なし）。
i 斜体（強調なし）。
u 下線。
mark ハイライト表示。
small 小さい文字。

リスト

ul 順序なしリスト。
ol 順序ありリスト。
li リスト項目。

リンクとメディア

a ハイパーリンクを定義。
img 画像を挿入。
video 動画を埋め込み。
audio 音声を埋め込み。
source メディアのソースを指定。
iframe 別ページを埋め込み。

テーブル

tr 行を定義
th 見出しセル。
td 通常のセル。
thead 表のヘッダー部分。
tbody 表の本体部分。
tfoot 表のフッター部分。
caption 表のタイトル。

フォーム

form フォームを定義。
input 入力フィールド（テキスト、パスワード、チェックボックスなど）。
textarea 複数行の入力フィールド。
button ボタンを定義。
select ドロップダウンメニュー。
option ドロップダウンメニューの項目。
label フォーム要素のラベル。
fieldset フォーム要素をグループ化。
legend グループのタイトル。

その他

script JavaScriptコードを埋め込み。
noscript JavaScriptが無効な場合に表示される内容。
canvas 描画領域を定義（JavaScriptで使用）。
svg ベクター画像を描画。
